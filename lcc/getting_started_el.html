Συνηθισμένα σενάρια

Αυτόματη ενέργεια κλικ σε ένα κουμπί

Όταν εμφανιστεί το παράθυρο που περιέχει το κουμπί, τοποθετήστε το δείκτη του ποντικιού στο πλήκτρο και πατήστε το πλήκτρο συντόμευσης του ClickOff στο πληκτρολόγιό σας (προεπιλογή: Ctrl+Alt+D). Μπορείτε να κάνετε αυτή τη διαδικασία και με τα πλαίσια ελέγχου.

Αυτόματο κλείσιμο ενός παραθύρου

Όταν εμφανιστεί το παράθυρο που θέλετε να κλείσει, τοποθετήστε το δείκτη του ποντικιού σας στη γραμμή τίτλου και πατήστε το πλήκτρο συντόμευσης ClickOff στο πληκτρολόγιό σας (προεπιλογή: Ctrl+Alt+D).

Αυτόματη συμπλήρωση σε ένα κείμενο

Όταν το πεδίο κειμένου είναι στην οθόνη σας, κάντε διπλό κλικ στο εικονίδιο του ClickOff για να ανοίξετε τη λίστα ClickOff. Στη συνέχεια τοποθετήστε το ποντίκι σας πάνω στο πεδίο κειμένου και πατήστε το πλήκτρο συντόμευσης ClickOff στο πληκτρολόγιό σας (προεπιλογή: Ctrl+Alt+D). Κάντε διπλό κλικ στη νέα καταχώρηση (στο κάτω μέρος της λίστας) και πληκτρολογήστε το κείμενο που πρέπει να συμπληρωθεί στο πεδίο κειμένου Συμπλήρ.:.
Σας συνιστώ ιδιαίτερα να μη χρησιμοποιείτε αυτή τη λειτουργία για να συμπληρώνετε τους κωδικούς πρόσβασης, γιατί η αποθήκευση των δεδομένων γίνεται χωρίς κρυπτογράφηση στη λίστα ClickOff!

Προβλήματα

Σε ένα κουμπί δεν γίνεται κλικ / ένα πεδίο κειμένου δεν συμπληρώνεται

Πιθανές αιτίες:
  1. Το κουμπί/πεδίο κειμένου δεν έχει ληφθεί από τις τυποποιημένες βιβλιοθήκες των Windows αλλά είναι μια άλλη δημιουργία (π.χ. του Firefox).
    Λύση: Ξεκινώντας από την έκδοση 1.83 το ClickOff μπορεί να χειριστεί αυτά τα κουμπιά. Κάτι τέτοιο γίνεται απλά, κάνοντας κλικ σε μια σταθερή θέση μέσα στο παράθυρο. Παρακαλώ να είστε προσεκτικοί εδώ! Το ClickOff θα κάνει κλικ σε αυτή τη θέση, σε κάθε παράθυρο που έχει τον ίδιο τίτλο! Αυτό μπορεί να έχει ανεπιθύμητες επιπτώσεις..
  2. Το κουμπί επικαλύπτεται αόρατα από ένα περίγραμμα. Το ClickOff τότε θα εντοπίσει τα όρια αντί για το κουμπί και στα όρια φυσικά, δεν μπορεί να κάνει κλικ.
    Λύση: Εισάγετε τα κείμενα κουμπιών με το χέρι στις ιδιότητες εγγραφών. Παρακαλώ σημειώστε ότι το υπογραμμισμένο γράμμα δημιουργείται χρησιμοποιώντας το σύμβολο &. Έτσι, για παράδειγμα, εάν ένα κείμενο κουμπιού είναι Ακύρωση πρέπει να εισάγετε Α&κύρωση.

Συνήθεις ερωτήσεις

  1. Ερ: Το ClickOff δεν λειτουργεί με τα Windows Vista/Windows 7;
    Απ: Έχω αναφορές που λένε, τόσο ότι λειτουργεί όσο και ότι δεν λειτουργεί. Δεν ξέρω, η πιο πρόσφατη έκδοση των windows που χρησιμοποιώ είναι Windows XP σε ένα netbook. Όλα τα άλλα μηχανήματά μου λειτουργούν σε Linux. Απλώς δοκιμάστε το, δεν πρόκειται να σπάσει κάτι.
  2. Ερ: Πού πήγε ο οδηγός;
    Απ: Έχει εγκαταλειφθεί υπέρ της μεθόδου του πλήκτρου συντόμευσης. Πολλοί χρήστες παραπονέθηκαν για το μεγάλο χρονικό διάστημα που έπρεπε να κάνουν κλικ και για τη διαδικασία πληκτρολόγησης για προσθήκη νέου κουμπιού, κι έτσι προσπάθησα να προσφέρω κάτι απλούστερο.
  3. Ερ: Στο παράθυρο διαλόγου «Προηγμένα χαρακτηριστικά» υπήρχε ένα κουμπί «Ανίχνευση κειμένου». Που πήγε αυτό και πώς μπορώ να ξανακάνω ανίχνευση κειμένου χωρίς αυτό;
    Απ: Αυτά επίσης έχουν αντικατασταθεί από το πλήκτρο συντόμευσης. Τώρα αν θέλετε να ξανακάνετε ανίχνευση κειμένου, τοποθετήστε απλώς το δείκτη στο κείμενο και πατήστε το πλήκτρο συντόμευσης - αυτό ήταν!
  4. Ερ: Πού βρίσκεται το αρχείο της λίστας;
    Απ: Όπου αποφασίσατε όταν κάνατε εγκατάσταση το ClickOff. Αυτό είτε βρίσκεται στον προσωπικό σας κατάλογο δεδομένων εφαρμογών εφαρμογών (προεπιλογή: c:\documents and settings\(το όνομά σας)\application data\clickoff) ή στην καθολική (default: c:\documents and settings\All Users\application data\clickoff). Εάν δεν είναι σε καμία από τις θέσεις αυτές, το ClickOff θα εξετάσει επίσης τον κατάλογο εκκίνησης (προεπιλογή: c:\program files\clickoff)
  5. Ερ: Το ClickOff κολλάει / δεν ξεκινάει!
    Απ: Συνήθως υπάρχει ένα σφάλμα στο αρχείο λίστας ή στο config. Ακόμη και αν έχω επιχειρήσει να βγάλω όσα περισσότερα σφάλματα μπορούσα, ίσως να έχω χάσει ορισμένα παράλογα σενάρια. Έτσι αν μου γράψετε σχετικά με τις περιπτώσεις κατάρρευσης του ClickOff, παρακαλώ να επισυνάψετε το αρχείο λίστας «clickoff.lst» (βλ. παραπάνω σχετικά με το πώς μπορείτε να το βρείτε) και το αρχείο «clickoff.cfg», για να τα ελέγξω. Παρακαλώ μην επισυνάψετε ενδείξεων σφαλμάτων των windows, δεν μπορούν να βοηθήσουν ακόμη.
